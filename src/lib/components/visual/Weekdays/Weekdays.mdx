`Weekdays` component displays row with weekday names on the top of day calendar. It receives collection of localised weekday names.
```json
{
  "short": "su",
  "narrow": "s",
  "abbreviated": "Sun",
  "wide": "Sunday",
  "numeric": 0
}
```
 Replace it with custom component.

<Playground>
  {() => {
    const style = {
      boxSizing: 'border-box',
      flex: '1 1 calc(100% / 7)',
      color: 'black',
      textAlign: 'center',
      padding: '0',
      border: '2px dashed pink',
      margin: '2px',
      fontSize: '12px',
      height: '24px',
      lineHeight: '24px'
    };
    const CustomComponent = ({name}) => {
      return (
        <div style={style}>
          {name.narrow}
        </div>
      )
    };
    return (
      <Datepicker
        onDateSet={date => {console.log('date set', date);}}
        minPrecision="day"
        renderWeekDayAs={CustomComponent}
        title="Demo datepicker" />
    )
  }}
</Playground>
